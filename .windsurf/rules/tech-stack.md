React-boilerplate is a module react application build with the following stack: 
- React.js
- Redux // for state management
- RTK (Redux Tool Kit) // for handling state slices and mutations
- RTK Query // for API calls and managing data through slices in redux store
- ShadCN/UI // for presentational UI components
- TailwindCSS // for CSS and styling
- React-Router-DOM // for dynamic and static routing
- TipTap // for creating WSIWYG editor
- @clerk/clerk-react // for authentication
- il8next // for multi-lingual support
- lucide-react // for icons
- react-markdown // for rendering markdown
- react-helmet-sync // for handling head tags (page title, keywords, description etc.)
- zod // for schema validation for forms
- react-hook-form // for form creation and validation using Zod
- sonner // for triggering notifications on the UI
- date-fns // for manipulating dates
